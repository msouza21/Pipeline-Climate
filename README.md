# Pipeline-Climate

Information  

This is a learning project of data engineering with purpose develop data skills, using APIs to request data and testing the workflow/infrastructure  
Using Apache Tools in docker conteiner - Spark, Airflow, Kafka, Flink and Database tools as PostgreSQL and MongoDB with support of DuckDB for be a 'Data Warehouse'  
Where the folders BGD is abreviated of Big Data and each of the BGD-* directories contains a docker-compose.yml that defines services related to the respective tool.  
To use these services with the pipeline-climate project, you must configure volumes and other options according to the project's needs.  

Settings 

When cloning this repository, it is important to configure volumes and other options according to your local environment. The docker-compose.yml files contained in the BGD-* folders serve as examples and should be adapted as necessary.
